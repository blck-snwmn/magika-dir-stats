# magika-dir-stats

A collection of tools to analyze directories and aggregate file statistics by type using [Magika](https://github.com/google/magika).

## Tools

- **python/**: Python-based CLI tool for analyzing file types and aggregating capacity
- **rust/**: Rust-based CLI tool with the same functionality

## Requirements

Each tool has its own dependencies. See the respective directory's README for details.
